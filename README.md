# Pyro Engine ![](https://img.shields.io/github/license/RottenThunder/PyroEngine "Pyro Engine is licensed under the AGPL-3.0 License")

A Specialised 2D Game Engine

Pyro Engine is a game engine designed for efficient 2D game development in C++, it provides all the common tools that are neccessary for developers aswell as features that are considered to be rare and unique. Optimised as can be and extremly versatile, the engine will not be a hinderance; it will be a time saver. The level of support and control that the engine offers is a useful asset for all aspiring and veteren developers out there.

The making of the engine has been set to use as little third-party libraries as possible, this way all the source code is hand-written and hence anyone can understand what it means and does unlike most third-party libraries where it is really hard to read and understand.

***

In order to setup the project, perform a regular `git clone https://github.com/RottenThunder/PyroEngine`, recursive cloning is not neccessary. Then run the [Windows_Setup.bat](https://github.com/RottenThunder/PyroEngine/blob/master/Windows_Setup.bat) file, this will generate a Visual Studio solution file from where you can edit and run the code. So far the engine only has a Windows build, but once more platforms are supported there will be further information in this section.
